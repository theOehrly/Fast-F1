API Reference

This page gives an overview of all public FastF1 objects, functions and methods.
All classes and functions exposed in ``fastf1.*`` namespace are public.

The following subpackages are public.

- ``fastf1.core``
- ``fastf1.ergast``
- ``fastf1.events``
- ``fastf1.legacy``
- ``fastf1.livetiming``
- ``fastf1.plotting``
- ``fastf1.utils``

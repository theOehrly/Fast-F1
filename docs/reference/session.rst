Session
=======

.. currentmodule:: fastf1.core

The :class:`Session` is a core part of FastF1 and serves as a starting point
for accessing various data. A :class:`Session` object represents a specific
race, qualifying, sprint or other Formula 1 session and all data related to
such a session can be accessed through it.
